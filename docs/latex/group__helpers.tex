\doxysubsection{Detailed Description}
EVMC Helpers. 

A collection of C helper functions for invoking a VM instance methods. These are convenient for languages where invoking function pointers is \char`\"{}ugly\char`\"{} or impossible (such as Go).

It also contains helpers (overloaded operators) for using EVMC types effectively in C++.
