The recipient of the message. 

This is the address of the account which storage/balance/nonce is going to be modified by the message execution. In case of \mbox{\hyperlink{group__EVMC_ggab2fa68a92a6828064a61e46060abc634abcf3ae29d9a88ff70b98374fc665694a}{EVMC\+\_\+\+CALL}}, this is also the account where the message value \mbox{\hyperlink{structevmc__message_adc7e2442cb362417931524ee904ecb33}{evmc\+\_\+message\+::value}} is going to be transferred.
