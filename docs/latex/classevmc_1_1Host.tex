\doxysubsection{Detailed Description}
Abstract class to be used by \mbox{\hyperlink{classevmc_1_1Host}{Host}} implementations. 

When implementing EVMC \mbox{\hyperlink{classevmc_1_1Host}{Host}}, you can directly inherit from the \mbox{\hyperlink{classevmc_1_1Host}{evmc\+::\+Host}} class. This way your implementation will be simpler by avoiding manual handling of the \mbox{\hyperlink{structevmc__host__context}{evmc\+\_\+host\+\_\+context}} and the \mbox{\hyperlink{structevmc__host__interface}{evmc\+\_\+host\+\_\+interface}}.
