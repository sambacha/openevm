\doxysubsection{Detailed Description}
Helpers for optional storage of \mbox{\hyperlink{structevmc__result}{evmc\+\_\+result}}. 

In some contexts (i.\+e. \mbox{\hyperlink{structevmc__result_ae7f37ff7c9cfa3b58af2c89f4d610522}{evmc\+\_\+result\+::create\+\_\+address}} is unused) objects of type \mbox{\hyperlink{structevmc__result}{evmc\+\_\+result}} contains a memory storage that MAY be used by the object owner. This group defines helper types and functions for accessing the optional storage.
